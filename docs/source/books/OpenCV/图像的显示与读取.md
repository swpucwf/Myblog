### 图像的读取举例

```python
import cv2
def show_img():
    img =cv2.imread("../images/img.png")
    print(type(img))
    print("图像形状:",img.shape)
    cv2.imshow("lisa",img)
    cv2.waitKey(1000)
    cv2.destroyAllWindows()

if __name__ == '__main__':
    show_img()
```

#include <opencv2/opencv.hpp>
